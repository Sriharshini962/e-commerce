# 🛍️ UrbanMart - Complete E-Commerce Website

UrbanMart is a fully responsive e-commerce web application that allows users to browse products, add items to cart, register/login, and contact the store. This project demonstrates a complete online shopping experience with both frontend and backend functionality.

## ✨ Features

### 👤 User Management
- User Registration with validation
- User Login with session management
- Password protection
- Logout functionality

### 🛒 Shopping Features
- Product catalog with categories
- Add to cart functionality
- Update cart quantities
- Remove items from cart
- Cart total calculation
- Local storage cart persistence

### 📄 Pages Included
- Home Page with featured products
- Products Page with all items
- Services Page
- Contact Page with form submission
- Cart Page
- Login Page
- Registration Page

### 🎨 Design
- Modern, clean interface
- Black, White, Orange color scheme
- Responsive design (works on mobile, tablet, desktop)
- Font Awesome icons
- Smooth animations and hover effects

### 💾 Database
- MySQL database with 3 tables (users, products, contacts)
- 12+ sample products pre-loaded
- Secure password storage
- Contact message storage

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP / Apache
- **Icons:** Font Awesome 6
- **Fonts:** Google Fonts (Inter)
